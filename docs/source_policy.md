# 소스 사용 정책 (저작권·인용 기준)

## 허용 소스 유형

| 유형 | rights_status | 인용 한도 | 처리 방식 |
|---|---|---|---|
| 공식 문서·보도자료 | free | 전문 요약 가능 | 링크 포함 필수 |
| 뉴스 기사 | restricted | 200자 이내 | 링크 + 해설 중심 |
| 학술 논문 | paid/restricted | 핵심 수치·결론만 | 출처 표기 필수 |
| 인터뷰 발언 | free (동의 시) | 발언 전문 가능 | 비식별화 검토 필수 |
| 이미지·그래프 | 별도 확인 | 원칙적으로 직접 사용 금지 | 라이선스 확보 후 사용 |

## 금지 행위

- 뉴스 기사의 본문 전체 번역 및 게시 (원문 대체형 침해)
- 출처 명시 없는 수치·통계 사용
- AI가 생성한 가상 기업 사례·사람 이름 사용
- 원저작자 동의 없는 인터뷰 전문 게시

## 소스 등록 필수 필드

source_registry.py에 소스 등록 시 반드시 입력:
- `url`: 원문 URL
- `publisher`: 발행사/저자명
- `rights_status`: free / restricted / paid / unknown
- `quote_limit`: 인용 가능 글자 수 (기본 200자)
- `article_id`: 연결된 기사 ID
