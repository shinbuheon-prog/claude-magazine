# 편집 검수 체크리스트

Ghost Admin에서 게시 전 반드시 통과해야 하는 10개 항목.

## 필수 확인 항목

- [ ] **출처 연결** — 핵심 주장마다 source_id가 연결되어 있는가
- [ ] **원문 대체 방지** — 번역·요약이 원문을 대체하지 않고 해설·분석이 추가되었는가
- [ ] **제목 일치성** — 제목과 본문 사이에 과장·낚시성 불일치가 없는가
- [ ] **사실 일치** — 인용문, 수치, 사례명이 원문과 일치하는가
- [ ] **AI 환각 방지** — AI가 생성한 가상 기업 사례·수치가 섞이지 않았는가
- [ ] **비식별화** — 인터뷰 발언, 익명 제보에 대한 비식별화가 끝났는가
- [ ] **시각물 라이선스** — 이미지·표·차트의 원천 데이터와 라이선스가 기록되었는가
- [ ] **AI 고지** — AI 보조 작성/분석 사용 고지 문구 필요 여부를 검토했는가
- [ ] **정정 책임자** — 게시 후 정정 책임자와 1차 응답 기한(24시간)이 지정되었는가
- [ ] **로그 저장** — 모든 Claude API 호출의 request_id가 logs/에 저장되었는가

## 팩트체크 판정 기준 (참고)

| 판정 | 의미 | 처리 |
|---|---|---|
| 확인됨 | 출처에서 직접 확인 | 통과 |
| 과장됨 | 출처 근거 있으나 표현 과장 | 표현 수정 후 통과 |
| 출처 불충분 | 출처가 주장을 명확히 지지 않음 | 추가 출처 확보 또는 삭제 |
| 수정 필요 | 출처와 불일치·사실 오류 | 반드시 수정 |

## 정정 프로토콜

1. 독자 제보 또는 내부 발견 시 **24시간 내 1차 응답**
2. 기사 상단에 정정 고지 추가 (날짜·내용·정정 이유)
3. source_registry에서 해당 source_id 권리 상태 재검토
4. 중대 오류 시 소셜 채널에도 정정 공지
