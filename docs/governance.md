## 발행사·관계사 서비스 소개 코너 (Sponsored Content)

본 매거진은 발행처(클래스메소드 그룹)와 관계사·사내 커뮤니티 활동을 정기/비정기 코너로 게재할 수 있다. 무료 발행 매체의 신뢰성을 보호하기 위해 다음 6개 항목을 의무 준수한다.

| 항목 | 표기 의무 |
|---|---|
| 1. 코너 표지 우상단 배지 | "AD" 또는 "Sponsored" 명시 (지름 24px 이상, 본문 색상 대비 명확) |
| 2. 첫 페이지 footer 1줄 고지 | "본 코너는 발행사 클래스메소드 그룹 또는 관계사의 서비스/활동 소개입니다" |
| 3. 시각 분리 | 본문 기사·외부 인용·자가 사례와 디자인 톤 차별화 (예: 발행사 코퍼레이트 톤 그대로 임베드) |
| 4. 발행 분량 비율 상한 | 매월 80p 기준 광고 페이지 **10% 이하** (= 8p 이하) |
| 5. 광고성 콘텐츠와 일반 기사 식별성 | 본문 기사 중 발행사 서비스 언급 시 1줄 footnote에 "참고: 발행사 서비스" 명시 |
| 6. Colophon 명시 | 매호 Colophon에 "본 호 광고 페이지 N건 (P페이지)" 1줄 표기 |

**예외 사항**:
- 매거진 자체 운영 회고(예: "Cowork SOP 한 달 사례")는 광고 코너로 분류하지 않음. 단 본문에 "본 사례는 매거진 자체 운영 회고이며 발행사 서비스 홍보가 목적이 아님" 명시 권장.
- 사외 커뮤니티 안내(예: r/ClaudeAI, claudecode.co.kr 등 외부 커뮤니티)는 광고 코너로 분류하지 않음. 단 발행사 주최 커뮤니티(예: 클래스메소드코리아 Claude 밋업)는 광고 코너로 분류.

**검증**:
- 매호 발행 직전 `publish-gate` skill의 표준 체크리스트에 "Sponsored Content 6항목 준수 확인" 추가
- 위반 발견 시 발행 보류 + 편집장 명시 승인 후 발행
