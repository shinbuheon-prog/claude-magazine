| 항목 | 처리 기준 |
|---|---|
| 처리 목적 | 뉴스레터 발송, 독자 분석 (명시 필수) |
| 보유 기간 | 구독 해지 후 30일 내 삭제 |
| 제3자 제공 | 원칙 금지 (서비스 운영 필수 파트너 제외) |
| 해외 이전 | Anthropic API 전송 시 비식별화 데이터만 허용 |
| 삭제 절차 | 이메일 요청 → 7영업일 내 처리 |

**Claude에 보내는 데이터**: 내부 메모, 인터뷰 녹취 전송 시 반드시 사전 비식별화.
