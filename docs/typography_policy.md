# Typography Policy

## Format Policy

- Magazine web/PDF:
  - Primary font: `Noto Serif KR`
  - Use for cover, long-form article body, insight narratives
- SNS card news:
  - Primary font: `Pretendard`, fallback `Noto Sans KR`
  - Use for 1080x1350 card assets and caption-oriented visual layouts
- Numeric/source markers:
  - `JetBrains Mono`

## Guardrails

- Do not mix serif body typography into SNS card slides.
- Do not use SNS sans tokens as the default for magazine pages.
- SNS card text should keep line-height `1.4~1.6` and letter-spacing `-0.02em`.
- Primary SNS text contrast should stay in `#111111 ~ #333333`.
